# TACHYON MVP v1

## Must work

- Build on Windows MSVC
- Build on Linux GCC
- Run TachyonSmokeTests
- Author scene through C++ Builder API
- Compile SceneSpec deterministically
- Render PNG sequence
- Render MP4 through FFmpeg
- Produce repeatable output hash for canonical scenes

## Not in MVP

- ProRes
- GUI preview window
- Vulkan backend
- OIDN
- Full 3D production renderer
- Advanced tracker features
- Browser/web renderer
- New editor/UI
