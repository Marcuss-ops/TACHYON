## What This Baseline Covers

- Core build and test entrypoints.
- Scene authoring primitives and the current build graph.

- 3D, tracker, audio mux, and catalog tooling remain partially gated or in transition.
- The repository still has open cleanup and architecture debt tracked in GitHub issues.
